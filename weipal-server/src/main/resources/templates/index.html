Weipal

A painless self-hosted Weipal service

Easy to install

Simply run the binary for your platform. Or ship Gogs with Docker or Vagrant, or get it packaged.

Cross-platform

Weipal runs anywhere Java can compile for: Windows, Mac OS X, Linux, ARM, etc. Choose the one you love!

Lightweight

Weipal has low minimal requirements and can run on an inexpensive Raspberry Pi. Save your machine energy!

Open Source

It's all on Git! Join us by contributing to make this project even better. Don't be shy to be a contributor!
